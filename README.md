# Jordan Schilling Portfolio

Personal portfolio site built with Jekyll and published with GitHub Pages.

## Local Development

1. Install Ruby and Bundler.
2. Install dependencies:

	bundle install

3. Start the site locally:

	bundle exec jekyll serve

4. Open http://127.0.0.1:4000

## Content Structure

- Main pages are in the repository root as Markdown files.
- Posts are stored in _posts.
- Layouts and includes are in _layouts and _includes.
- Styling is in _sass and assets/css.

## Public Repo Safety Checklist

- Do not commit secrets, tokens, API keys, or private certificates.
- Use sample values in code snippets.
- Keep local editor/history folders out of git.
- Review new content before publishing.
