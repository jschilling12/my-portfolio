## Technical Focus

My technical focus is centered on automation and reliability engineering. The core disciplines include:

**Automation Frameworks**: I design and build test automation frameworks using industry-standard patterns. The Python & Playwright framework uses the Page Object Model and object-oriented principles to create maintainable, scalable test suites — applied to real projects with real CI/CD integrations.

**DevOps Engineering**: I approach DevOps as an engineering discipline, not a job title. This means building infrastructure as code, automating deployments, monitoring systems, and treating operational concerns as first-class engineering requirements.

**System-Level Software**: I build tools that interact with operating systems at a low level — monitoring processes, tracking application focus time, and providing diagnostic information about system health. The FocusBuddy project is the primary expression of this interest.
